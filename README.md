# 공밥 사장님용 앱 (Gongbab Owner)

공밥 사장님 전용 앱입니다. 사장님들은 이 앱을 통해 일일 식사 현황, 업체별 식사 내역, 월별 정산 등을 확인할 수 있습니다.
